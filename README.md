# Telegram-бот для удаления фона с фотографий

Этот проект представляет собой Telegram-бота, который удаляет фон с фотографий при помощи библиотеки [rembg](https://github.com/danielgatis/rembg). Бот ведёт учёт пользователей и количества обработанных ими изображений в базе данных SQLite.

## Возможности

- Удаление фона из фотографий и возврат результата в формате PNG с прозрачным фоном.
- Учёт пользователей с сохранением имени, username, даты регистрации и количества обработанных фото.
- Интерактивное меню с кнопками для быстрого доступа к основным функциям.
- Команда `/start` открывает главное меню, `/help` выводит инструкцию, `/profile` показывает статистику пользователя.

## Требования

- Python 3.8+
- Созданный Telegram-бот и полученный токен доступа (уже указан в `config.py`).

## Установка и запуск

1. Клонируйте репозиторий и перейдите в директорию проекта.
2. Создайте виртуальное окружение (рекомендуется):

   ```bash
   python -m venv venv
   source venv/bin/activate  # macOS/Linux
   venv\Scripts\activate     # Windows
   ```

3. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

4. Запустите бота:

   ```bash
   python bot.py
   ```

## Структура проекта

- `bot.py` — основной файл с логикой Telegram-бота.
- `config.py` — конфигурация проекта и тексты сообщений.
- `database.py` — работа с базой данных SQLite.
- `requirements.txt` — список зависимостей проекта.
- `README.md` — документация по проекту.

## Примечания

- База данных создаётся автоматически при первом запуске и хранится в файле `bot_database.db`.
- Для корректной обработки фотографий библиотека `rembg` загружает необходимые модели при первом запуске.
