# Telegram-бот для удаления фона с фотографий

Этот проект представляет собой Telegram-бота, который удаляет фон с фотографий при помощи облачного сервиса [remove.bg](https://www.remove.bg/). Бот ведёт учёт пользователей и количества обработанных ими изображений в базе данных SQLite и полностью готов к запуску в Termux.

## Возможности

- Удаление фона из фотографий через API remove.bg с возвратом результата в формате PNG с прозрачным фоном.
- Учёт пользователей с сохранением имени, username, даты регистрации и количества обработанных фото.
- Интерактивное меню с кнопками для быстрого доступа к основным функциям.
- Команда `/start` открывает главное меню, `/help` выводит инструкцию, `/profile` показывает статистику пользователя.

## Требования

- Python 3.8+
- Termux с установленным пакетом `python`
- Доступ в интернет для обращения к remove.bg

## Установка и запуск в Termux

```bash
pkg install python
git clone <url-репозитория>
cd <папка-репозитория>
pip install --upgrade pip
pip install -r requirements.txt
python bot.py
```

После запуска бот начнёт принимать команды и фотографии, фон будет удаляться автоматически.

## Структура проекта

- `bot.py` — основной файл с логикой Telegram-бота.
- `config.py` — конфигурация проекта (токены и тексты сообщений).
- `database.py` — работа с базой данных SQLite.
- `requirements.txt` — минимальный список зависимостей.
- `README.md` — документация по проекту.
